OpenNEX on the Web

Source Code

The source is available on github at:

https://github.com/alexmilowski/opennex

There are full instructions for running the code at the github page.

Live Demonstration

A live demonstration of the use of the data is available at:

http://data.pantabular.org/opennex/explore/

with the data hosted at http://data.pantabular.org/opennex/data/ where:

  • ./{name}/{yearMonth}/{resolution}/ is the summary for data set name (e.g., avg-rcp85, quartile75, quartile50, or quartile25) for year/month yearMonth (e.g., 2014-10) for the resolution (e.g., 30, 60, or 120)
  • ./{name}/{yearMonth}/{resolution}/{partition} is the raw data partitioned by quadrangle with sequence number partition

All data is returned as HTML with RDFa annotations.

AMI for Running Demo

AMI ami-8a5bdfe2 is configured to run the Web application accessing the data directly in S3. You can run this server on any instance type by m3.medium is more than sufficient. Once the server has started, just access the Website at the root of the server on the default port (i.e., port 80). The first time you access the server, it compiles pipelines and bytecode and so the very first time you access the server it may take longer than subsequent times.
